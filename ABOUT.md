# Sobre o c_valor_intrinseco

## O que é?

O **c_valor_intrinseco** é uma ferramenta de análise financeira desenvolvida para auxiliar investidores no cálculo do valor intrínseco de empresas listadas em bolsa de valores.

## Por que foi criado?

O investimento em valor (*value investing*) requer a avaliação precisa de empresas usando múltiplos modelos financeiros. Cada modelo tem suas vantagens e limitações:

- **Gordon** é ideal para empresas com dividendos estáveis
- **DCF** funciona melhor para empresas com fluxo de caixa previsível
- **Graham Number** é conservador e focado em segurança
- **Owner Earnings** (Buffett) prioriza o caixa real disponível

Nenhum modelo sozinho é suficiente. A **combinação de vários modelos** oferece uma visão mais completa e reduz o risco de erro.

## Objetivos

1. **Educacional**: Ensinar os principais modelos de valuation
2. **Prático**: Fornecer uma ferramenta útil para decisões de investimento
3. **Acessível**: Ser simples de usar, mesmo para iniciantes
4. **Abrangente**: Cobrir os modelos mais utilizados por investidores profissionais

## Modelo Mental

A ferramenta segue a filosofia de investimento de value investing:

> "O preço é o que você paga. O valor é o que você recebe." — Warren Buffett

Nenhum modelo é perfeito. O objetivo é ter uma **faixa de valores justos** e tomar decisões com **margem de segurança**.

## Limitações

- Os dados do Yahoo Finance podem estar desatualizados
- Os modelos dependem de premissas que podem estar erradas
- Não considera fatores qualitativos (gestão, moat, ciclo)
- Não é recomendação de investimento
